# Appiness website

Minimal static website for Appiness, ready for GitHub Pages.

## Files

- `index.html` — homepage
- `privacy.html` — Privacy Policy URL for App Store Connect
- `support.html` — Support URL for App Store Connect
- `terms.html` — Terms of Use
- `styles.css` — shared styling
- `assets/icon.png` — app icon

## Publish on GitHub Pages

1. Create a public GitHub repository, for example `appiness` or `appiness-site`.
2. Upload all files in this folder to the repository root.
3. In GitHub, open **Settings → Pages**.
4. Under **Build and deployment**, select:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
5. Save.
6. GitHub will publish the site at a URL like:
   - `https://YOUR_USERNAME.github.io/appiness/`

## App Store Connect URLs

Use:

- Privacy Policy URL: `https://YOUR_USERNAME.github.io/appiness/privacy.html`
- Support URL: `https://YOUR_USERNAME.github.io/appiness/support.html`
- Terms URL, if needed: `https://YOUR_USERNAME.github.io/appiness/terms.html`

## Before publishing

Replace `support@appiness.app` with your real support email if needed.
